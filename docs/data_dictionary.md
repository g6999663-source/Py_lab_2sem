# Data Dictionary – mart витрина (variant_04, London)

- **Версия:** 1.0
- **Дата:** 2026-05-31
- **Слой:** mart (суточная витрина)
- **Источник:** `data/mart/variant_04/daily_weather.csv`

---

## Описание набора данных

Витрина содержит агрегированные метеорологические данные для Лондона за каждый день. Данные получены из Open‑Meteo Archive API и охватывают период с 1 по 14 января 2024 года. Одна строка = один день наблюдений.

**Бизнес-контекст:** анализ погодных условий в Лондоне, выявление трендов, аномалий и взаимосвязей между метриками.

---

## Описание колонок

### date

| Свойство     | Значение                     |
|--------------|------------------------------|
| **Тип**      | date                         |
| **Единица**  | календарный день             |
| **Nullable** | нет                          |
| **Описание** | Дата наблюдения в формате YYYY-MM-DD |
| **Откуда берётся** | Из `ts` поля normalized слоя |
| **Пример**   | 2024-01-01                   |

### city_id

| Свойство     | Значение                     |
|--------------|------------------------------|
| **Тип**      | string                       |
| **Единица**  | -                            |
| **Nullable** | нет                          |
| **Описание** | Уникальный идентификатор города |
| **Откуда берётся** | Из конфига `variant_04.yml` |
| **Пример**   | GB_LON                       |

### avg_temp

| Свойство     | Значение                     |
|--------------|------------------------------|
| **Тип**      | float                        |
| **Единица**  | °C (градус Цельсия)          |
| **Nullable** | нет                          |
| **Описание** | Среднесуточная температура воздуха |
| **Откуда берётся** | AVG(temperature_2m) из normalized |
| **Пример**   | 6.5                          |
| **Интерпретация** | Чем выше значение, тем теплее день |

### min_temp

| Свойство     | Значение                     |
|--------------|------------------------------|
| **Тип**      | float                        |
| **Единица**  | °C                           |
| **Nullable** | нет                          |
| **Описание** | Минимальная температура за день (обычно ночью) |
| **Откуда берётся** | MIN(temperature_2m) из normalized |
| **Пример**   | 3.2                          |

### max_temp

| Свойство     | Значение                     |
|--------------|------------------------------|
| **Тип**      | float                        |
| **Единица**  | °C                           |
| **Nullable** | нет                          |
| **Описание** | Максимальная температура за день (обычно днём) |
| **Откуда берётся** | MAX(temperature_2m) из normalized |
| **Пример**   | 9.8                          |

### avg_humidity

| Свойство     | Значable                     |
|--------------|------------------------------|
| **Тип**      | float                        |
| **Единица**  | % (процент)                  |
| **Nullable** | нет                          |
| **Описание** | Среднесуточная относительная влажность |
| **Откуда берётся** | AVG(relative_humidity_2m) из normalized |
| **Пример**   | 78.5                         |
| **Интерпретация** | Высокая влажность (>80%) может указывать на дождливую или туманную погоду |

### total_precip

| Свойство     | Значение                     |
|--------------|------------------------------|
| **Тип**      | float                        |
| **Единица**  | мм (миллиметры)              |
| **Nullable** | нет                          |
| **Описание** | Суммарное количество осадков за день |
| **Откуда берётся** | SUM(precipitation) из normalized |
| **Пример**   | 5.2                          |
| **Интерпретация** | 0 мм = сухой день, >10 мм = дождливый день |

### avg_windspeed

| Свойство     | Значение                     |
|--------------|------------------------------|
| **Тип**      | float                        |
| **Единица**  | км/ч (километры в час)       |
| **Nullable** | нет                          |
| **Описание** | Среднесуточная скорость ветра |
| **Откуда берётся** | AVG(wind_speed_10m) из normalized |
| **Пример**   | 12.3                         |
| **Интерпретация** | >20 км/ч – ветрено, >50 км/ч – шторм |

## Примеры использования

### Какой был самый тёплый день?

```sql
SELECT date, avg_temp
FROM daily_weather
ORDER BY avg_temp DESC
LIMIT 1;
```

### Какова корреляция между влажностью и осадками?

```sql
SELECT 
    CORR(avg_humidity, total_precip) AS humidity_precip_correlation
FROM daily_weather;
```

### Топ-3 самых ветреных дня

```sql
SELECT date, avg_windspeed
FROM daily_weather
ORDER BY avg_windspeed DESC
LIMIT 3;
```

## Ограничения данных

- **Период данных:** всего 14 дней (недостаточно для климатических трендов)
- **География:** только Лондон (одна точка)
- **Точность:** зависит от качества метеостанций Open‑Meteo
- **Пропуски:** при отсутствии данных API может возвращать null (в нашем периоде пропусков нет)
